## Building

> Requirements :
> - CMake 3.16 or greater
> - GCC or Clang or any C++/C compilator
> - OpenGL 4.6

To configure the CMake Project: 

```cmake -B build [-G <Generator>]```

To Build

```cmake --build build```

> you can also use `cmake-gui` if you want.
